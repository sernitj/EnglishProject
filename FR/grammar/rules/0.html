Indiquer une quantité
  • SOME ou ANY ?
    • Tous les deux veulent dire 'du, de la, des'. Lequel choisir ?

    Some : dans les phrases affirmatives, dans les phrases interrogatives formulant une offre ou une demande polie.
    • I need some butter.
    • J'ai besoin de beurre.

    Any : dans les autres phrases interrogatives, dans les phrases négatives.
    • Could I have some water, please?
    • Pourrais-je avoir un peu d'eau, s'il vous plaît ?
  • MUCH ou MANY ?
    • Tous les deux veulent dire 'beaucoup'. Lequel choisir ?

    Much : il est suivi d'un nom au singulier.
    • There is much traffic.
    • Il y a beaucoup de circulation.

    Many : il est suivi d'un nom au pluriel.
    • There are many cars.
    • Il y a beaucoup de voitures.
